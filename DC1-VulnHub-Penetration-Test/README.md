# DC-1 VulnHub Penetration Test

## Overview
Penetration testing assessment performed on the DC-1 vulnerable machine involving enumeration, exploitation, and privilege escalation.

## Objectives
- Identify vulnerabilities
- Gain initial access
- Escalate privileges
- Document findings

## Tools Used
- Nmap
- Gobuster
- Burp Suite
- Kali Linux

## Skills Demonstrated
- Enumeration
- Web Application Testing
- Exploitation
- Privilege Escalation

## Key Findings
- SQL Injection
- Sensitive Information Disclosure
- Misconfigured Services
- Weak Privilege Controls
