# OWASP CRAPI Security Assessment

## Overview
Security assessment conducted on the OWASP Completely Ridiculous API (CRAPI) to identify API security vulnerabilities and insecure business logic flaws.

## Objectives
- Assess API security controls
- Identify authentication and authorization flaws
- Evaluate OWASP API Security Top 10 risks

## Tools Used
- Burp Suite
- Postman
- Kali Linux
- JWT.io

## Skills Demonstrated
- API Security Testing
- Broken Object Level Authorization Testing
- Authentication Testing
- Business Logic Testing

## Key Findings
- Broken Object Level Authorization (BOLA)
- Broken Authentication
- Excessive Data Exposure
- Improper Authorization Controls
