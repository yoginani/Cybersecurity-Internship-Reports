# Tempus Fugit 5.2 Penetration Test

## Overview
Penetration testing assessment conducted on the Tempus Fugit 5.2 vulnerable machine to identify security weaknesses and obtain administrative access.

## Objectives
- Service Enumeration
- Vulnerability Discovery
- Exploitation
- Privilege Escalation

## Tools Used
- Nmap
- Gobuster
- Burp Suite
- Kali Linux

## Skills Demonstrated
- Enumeration
- Web Exploitation
- Privilege Escalation
- Security Reporting

## Key Findings
- Misconfigured Services
- Web Application Vulnerabilities
- Authentication Weaknesses
- Local Privilege Escalation
