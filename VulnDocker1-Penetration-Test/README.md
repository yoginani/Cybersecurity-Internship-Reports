# VulnDocker 1 Penetration Test

## Overview
Security assessment of the VulnDocker 1 vulnerable environment focusing on web application vulnerabilities and privilege escalation techniques.

## Objectives
- Enumerate services
- Identify vulnerabilities
- Exploit discovered weaknesses
- Escalate privileges

## Tools Used
- Nmap
- Gobuster
- Burp Suite
- Kali Linux

## Skills Demonstrated
- Enumeration
- Web Application Security Testing
- Exploitation
- Post Exploitation

## Key Findings
- Web Application Vulnerabilities
- Weak Access Controls
- Misconfigured Services
- Privilege Escalation Opportunities
